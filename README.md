# Luke de Boer — Personal Website

My professional website. A single-page personal/portfolio site for **Luke de Boer**, Senior Software Engineer.

- **Style:** Neo-brutalist, inspired by [retroui.dev](https://retroui.dev) — Archivo Black + Space Grotesk, hard offset shadows, thick black borders, yellow accent.
- **Stack:** A self-contained `index.html` (no build step, no dependencies beyond Google Fonts).
- **Hosting:** Drop-in ready for GitHub Pages — just enable Pages on this repo.

## Run locally

Open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
